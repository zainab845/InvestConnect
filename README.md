🚀 InvestConnect
InvestConnect is a full-stack, virtual matchmaking platform designed to bridge the gap between ambitious startup founders and potential investors. It provides a secure, interactive environment for entrepreneurs to pitch their businesses and for investors to discover, filter, and connect with startups that align with their financial goals.

Built as the capstone project for the Database Systems Lab at FAST NUCES.

✨ Core Features
Role-Based Access Control: Distinct, secure portals for Business Owners, Investors, and System Administrators.

Algorithmic Matching Engine: Investors receive a dynamically filtered feed of startups based on strict minimum/maximum budget constraints and preferred industry categories.

Interactive Investor Feed: Users can curate a personal Watchlist (⭐), leave public comments (💬) on pitches, and upvote businesses (👍).

Secure File Handling: Business owners can upload and attach PDF pitch decks to their profiles.

Admin Moderation Dashboard: A dedicated queue for platform administrators to review, approve, and publish new startup pitches before they go live on the global feed.

Direct Connections: Built-in tools for investors to securely pull founder contact information and initiate email conversations.

🛠️ Tech Stack
Frontend:

React.js

React Router (for secure, client-side routing)

Vite

Backend:

Node.js

Express.js

Multer (for multipart form data and PDF file uploads)

Bcrypt (for secure password hashing)

Database:

Microsoft SQL Server

Engineered with 7 normalized relational tables, strict foreign key constraints, complex JOINs, and transactional queries.
